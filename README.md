# Suguru Interest News

個人用のニュースPWAです。  
`アニメ系 / 映画実写 / ロボティクス / フードテック` の4ジャンルを、Main切り替えで個別に見られる構成にしています。

## 概要

- 対象フォルダー: `C:\Users\sweet\projects\news-pwa`
- GitHub: [HaTsu-hatsushima/news](https://github.com/HaTsu-hatsushima/news)
- 旧フォルダー `C:\Users\sweet\codex-projects\news-pwa` は通常作業では使いません
- 全ジャンル混在の Main「すべて」は廃止済みです
- 初期表示は `アニメ系` 固定です

このアプリは、記事数を無理に増やすよりも、見たい記事だけを安定して拾うことを優先しています。

## 基本方針

- Mainごとにジャンルを分ける
- Category と Filter は、そのMainに合うものだけ表示する
- ジャンル混在の一覧は作らない
- 件数が少なくても精度を優先する
- Main / Category 切り替え時は Filter の `すべて` を自動選択する

## UI / 動作ルール

- Mainを押したら、そのジャンル専用の Category と Filter だけ表示します
- Main / Category 切り替え時は、Filter の `すべて` を自動選択します
- Filterを再度押した時も消灯せず `すべて` に戻ります
- 0件時は空白ではなく案内文を表示します
- キャッシュ更新が必要な修正では、`index.html` の script version と `sw.js` の cache名を更新します

## 現在の完成状態

このアプリは、次の方向で完成形に近づけています。

- Main / Category / Filter の役割を明確化
- ジャンルごとの取得元とFilterルールを固定
- 単一収集元時の混在問題を解消
- 件数表示をできるだけ実際の表示内容に寄せて調整
- 初心者でも「今どこを見ているか」が分かりやすい構成に整理

つまり、完璧に件数を増やすことよりも、`見たい記事だけを安定して見る` ための個人用ニュースアプリとして仕上げています。
